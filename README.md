동영상 Super resolution 웹 서비스
===================================
2018년 캡스톤디자인(1) 에서 진행하는 딥 러닝을 사용한 동영상 Super resolution 웹 서비스  

# 사용하는 것
* Nginx
* Django
* Pytorch
* PostgreSQL
* opencv
* opencv-python
* numpy

# 팀 화질구지
역할 | 팀원명
----|------
웹백앤드|김동환
웹프론트|안지수
동영상SR|이태균
